User Guide

📸 캡처 프로그램 사용 가이드

네이버 블로그·카페, 인스타그램 콘텐츠를 자동으로 캡처하고 Google Drive에 저장하는 도구입니다.

목차
설치
① 확장 프로그램 설치 ② 웹앱 접속
기능
③ 통합검색 자동캡처 ④ 컨텐츠 즉시캡처 ⑤ 크롭 설정 ⑥ Drive 설정 ⑦ 캡처 로그
참고
자주 묻는 질문
🚀 캡처 프로그램이란?
광고주 보고를 위해 SNS 콘텐츠를 캡처하고 Google Drive에 자동 저장하는 도구입니다.
크롬 확장 프로그램과 웹앱(관리 페이지) 두 가지를 함께 사용합니다.
📝 네이버 블로그 ☕ 네이버 카페 📸 인스타그램
⏱
통합검색 자동캡처
지정한 시간에 네이버 통합검색 결과를 자동으로 캡처합니다.
⚡
컨텐츠 즉시캡처
블로그·카페 게시물 URL을 등록하고 버튼 한 번으로 즉시 캡처합니다.
1
크롬 확장 프로그램 설치
처음 한 번만 설치하면 됩니다
1
확장 프로그램 폴더 받기
담당자에게 캡처 프로그램 폴더를 전달받아 내 PC의 원하는 위치에 저장합니다.
(예: C:\Users\내이름\Documents\캡처프로그램)
💡중요: 이 폴더는 확장이 실행되는 동안 계속 필요합니다. 설치 후 폴더를 삭제하거나 이동하면 확장이 작동하지 않습니다.
2
Chrome 확장 관리 페이지 열기
Chrome 브라우저를 열고 주소창에 아래를 입력한 뒤 Enter를 누릅니다.
chrome://extensions
3
개발자 모드 켜기
확장 관리 페이지 오른쪽 상단에 있는 개발자 모드 토글을 클릭해서 켭니다.
토글이 파란색으로 바뀌면 활성화된 것입니다.
⚠ 주의: 개발자 모드를 켜야만 폴더로 확장을 설치할 수 있습니다. 반드시 켜주세요.
4
확장 프로그램 폴더 불러오기
개발자 모드를 켜면 상단에 버튼 3개가 나타납니다.
왼쪽의 "압축해제된 확장 프로그램을 로드합니다" 버튼을 클릭합니다.
→ 파일 선택 창이 열리면 1단계에서 저장한 캡처 프로그램 폴더를 선택합니다.
→ 폴더를 선택한 뒤 "폴더 선택" 버튼을 클릭합니다.
→ 확장 목록에 "캡처 프로그램"가 나타나면 설치 완료입니다.
5
설치 확인
Chrome 주소창 오른쪽의 퍼즐 조각(🧩) 아이콘을 클릭하면 설치된 확장 목록이 나옵니다.
캡처 프로그램이 보이면 정상적으로 설치된 것입니다.
💡자주 사용한다면 캡처 프로그램 옆의 📌 핀 아이콘을 눌러 주소창 옆에 고정해두세요.
2
웹앱(관리 페이지) 접속
스케줄 설정과 캡처 실행은 여기서 합니다
1
웹앱 주소 접속
Chrome에서 아래 주소로 접속합니다.
https://jwoooooooooo.github.io/capture/
💡자주 쓰는 페이지이므로 즐겨찾기(Ctrl+D)에 추가해두면 편리합니다.
2
확장 연결 확인
페이지 오른쪽 상단에 ✅ 확장 연결됨이 표시되면 정상입니다.
⚠ 확장 프로그램 필요가 표시되면 아래를 확인하세요.
→ 확장이 설치됐는지 확인 (chrome://extensions에서 캡처 프로그램 활성화 여부 확인)
→ 페이지를 새로고침(F5) 해보세요. 확장이 처음 깨어나는 데 몇 초 걸릴 수 있습니다.
→ 그래도 안 되면 chrome://extensions에서 캡처 프로그램의 새로고침(↺) 버튼을 클릭 후 웹앱 새로고침
3
통합검색 자동캡처
지정한 시간에 네이버 검색 결과를 자동으로 캡처합니다
네이버 통합검색 URL을 등록하고 실행 시간을 설정하면, 매일 그 시간에 자동으로 캡처해서 Google Drive에 저장합니다.
주로 키워드별 블로그 검색 결과를 정기적으로 기록할 때 사용합니다.
1
네이버 검색 URL 준비
네이버에서 캡처할 키워드를 검색한 뒤 주소창의 URL을 복사합니다.
예: https://search.naver.com/search.naver?query=듀오락+유산균
💡검색 결과 화면에서 Ctrl+L을 누르면 주소창 전체가 선택됩니다. 그 상태에서 Ctrl+C로 복사하세요.
2
스케줄 추가
웹앱에서 "통합검색 자동캡처" 탭을 클릭한 뒤 "+ 스케줄 추가" 버튼을 클릭합니다.
3
스케줄 정보 입력
팝업 창에서 아래 항목을 입력합니다.
→ 스케줄 이름: 알아보기 쉬운 이름 입력 (예: 듀오락_블로그검색)
→ 실행 시간: 시간 입력 후 "+ 추가" 버튼 클릭. 여러 시간대 추가 가능 (예: 09:00, 18:00)
→ URL 추가: "+ URL 추가" 버튼 클릭 → URL 입력란에 복사한 네이버 검색 URL 붙여넣기 → 키워드란에 키워드 입력 (예: 듀오락)
→ URL은 여러 개 추가 가능합니다. 같은 스케줄에 여러 키워드를 묶어서 관리할 수 있습니다.
→ 입력 완료 후 "저장" 버튼을 클릭합니다.
4
스케줄 활성화 확인
저장된 스케줄 카드 왼쪽의 토글 스위치가 켜진(파란색) 상태여야 자동 실행됩니다.
꺼져 있으면 클릭해서 활성화합니다.
5
즉시 실행 (선택)
예약 시간을 기다리지 않고 지금 바로 실행하려면 스케줄 카드의 "▶ 지금 실행" 버튼을 클릭합니다.
버튼이 "⏳ 실행 중..."으로 바뀌고 "■ 중지" 버튼이 나타납니다.
→ 캡처 중에는 새 창이 자동으로 열렸다 닫히며 진행됩니다. 정상적인 동작입니다.
→ 중간에 멈추려면 "■ 중지" 버튼을 클릭합니다.
⚠ 주의: 자동캡처가 실행되는 동안 PC가 켜져 있어야 하고, Chrome이 실행 중이어야 합니다. PC가 꺼지면 예약된 시간에 실행되지 않습니다.
6
크롭 설정 (선택)
검색 결과의 특정 영역만 캡처하고 싶다면 크롭을 설정합니다.
스케줄 카드를 클릭해 펼치면 URL 목록이 나타납니다. 각 URL 오른쪽의 "크롭 설정" 버튼을 클릭하세요.
자세한 방법은 ⑤ 크롭 설정 섹션을 참고하세요.
4
컨텐츠 즉시캡처
블로그·카페 게시물을 버튼 한 번으로 캡처합니다
특정 게시물 URL들을 묶음으로 저장해두고, 버튼 하나로 모든 게시물을 순서대로 캡처합니다.
네이버 블로그, 네이버 카페 게시물에 주로 사용합니다.
1
캡처할 게시물 URL 수집
캡처할 네이버 블로그 또는 카페 게시물 페이지를 열고 주소창의 URL을 복사합니다.
여러 게시물이 있다면 URL을 메모장에 한 줄에 하나씩 정리해둡니다.
2
묶음 추가
웹앱에서 "컨텐츠 즉시캡처" 탭을 클릭한 뒤 "+ 묶음 추가" 버튼을 클릭합니다.
3
묶음 정보 입력
팝업 창에서 아래 항목을 입력합니다.
→ 묶음 이름: 알아보기 쉬운 이름 입력 (예: 듀오락_블로그포스팅)
→ URL 목록: 캡처할 게시물 URL을 입력란에 붙여넣습니다. 여러 개일 경우 한 줄에 하나씩 입력합니다.
→ 입력 완료 후 "저장" 버튼을 클릭합니다.
4
크롭 설정 (권장)
게시물의 본문 영역만 캡처하려면 크롭을 먼저 설정합니다.
묶음 카드의 "✂ 크롭 설정" 버튼을 클릭하세요.
자세한 방법은 ⑤ 크롭 설정 섹션을 참고하세요.
💡크롭을 설정하지 않으면 페이지 전체가 캡처됩니다. 게시물 본문만 필요하다면 크롭을 먼저 설정하는 것을 권장합니다.
5
즉시 캡처 실행
묶음 카드의 "▶ 즉시 캡처" 버튼을 클릭합니다.
등록된 URL을 순서대로 자동으로 열어 캡처하고 Google Drive에 저장합니다.
→ 캡처 중에는 새 창이 자동으로 열렸다 닫히며 진행됩니다. 정상적인 동작입니다.
→ 중간에 멈추려면 "■ 중지" 버튼을 클릭합니다.
→ 완료 후 결과는 "캡처 로그" 탭에서 확인할 수 있습니다.
5
크롭 설정
캡처 영역을 좌우로 잘라 원하는 부분만 저장합니다
크롭은 좌우 범위를 지정합니다. 페이지에서 클릭 2번으로 시작점과 끝점을 정합니다.
1
크롭 설정 버튼 클릭
스케줄 카드(자동캡처)의 경우 카드를 펼쳐 URL 옆 "크롭 설정" 버튼을 클릭합니다.
묶음 카드(즉시캡처)의 경우 카드의 "✂ 크롭 설정" 버튼을 클릭합니다.
2
크롭 탭이 자동으로 열림
해당 URL의 페이지가 새 탭으로 자동 열립니다.
페이지 상단에 파란색 가이드 문구가 나타납니다:
1번째 클릭: 시작 X위치 (왼쪽) | 2번째 클릭: 끝 X위치 (오른쪽) | ESC: 취소
3
1번째 클릭: 시작 위치 설정
캡처를 시작할 왼쪽 경계 위치를 클릭합니다.
클릭하면 가이드 문구가 "✅ 시작 위치 설정!"으로 바뀝니다.
💡마우스를 움직이면 파란 세로선이 따라옵니다. 이 선이 잘릴 위치입니다. 원하는 위치에서 클릭하세요.
4
2번째 클릭: 끝 위치 설정
캡처를 끝낼 오른쪽 경계 위치를 클릭합니다.
클릭하면 "✅ 크롭 영역이 저장되었습니다!" 문구가 나타나고 탭이 자동으로 닫힙니다.
5
웹앱에서 반영 확인
웹앱 탭으로 돌아오면 자동으로 업데이트됩니다.
묶음/스케줄 카드에 ✂ 크롭: XXX~XXXpx 형태로 표시되면 저장 완료입니다.
💡크롭 영역을 다시 설정하고 싶으면 같은 버튼을 다시 클릭하면 됩니다. 기존 영역이 하늘색으로 표시되며, 새로 클릭하면 덮어씌워집니다.
6
Drive 설정
캡처 이미지가 저장될 Google Drive를 연결합니다
1
Drive 설정 탭으로 이동
웹앱 상단 탭에서 "Drive 설정"을 클릭합니다.
2
Google Drive 연결
"Google Drive 연결" 버튼을 클릭합니다.
"Apps Script 연결됨 ✓"이 표시되면 연결 완료입니다.
💡Drive 연결은 처음 한 번만 설정하면 됩니다. 이후 캡처된 이미지는 자동으로 지정된 Drive 폴더에 저장됩니다.
7
캡처 로그
캡처 결과를 확인합니다
1
로그 확인
웹앱 상단 탭에서 "캡처 로그"를 클릭합니다.
최근 100건의 캡처 기록이 표시됩니다.
→ ✓ (초록) : 캡처 성공. 파일명이 표시됩니다.
→ ✕ (빨강) : 캡처 실패. 오류 메시지를 확인하세요.
2
Drive에서 파일 확인
캡처된 이미지는 Google Drive의 지정 폴더에 아래 형식으로 저장됩니다.
스케줄명_키워드_날짜시간.jpg
💡예: 듀오락모니터링_듀오락_20260403_1430.jpg
?
자주 묻는 질문
Q
캡처 중에 다른 작업을 해도 되나요?
가능합니다. 캡처는 백그라운드에서 진행됩니다. 다만 캡처 중에 창이 자동으로 열렸다 닫히므로 화면이 잠깐 전환될 수 있습니다.
Q
자동캡처 예약 시간에 PC가 꺼져 있으면 어떻게 되나요?
해당 시간의 캡처는 건너뜁니다. PC가 켜지면 다음 예약 시간에 정상 실행됩니다.
Q
캡처 이미지가 Drive에 저장되지 않아요.
Drive 설정 탭에서 "Apps Script 연결됨 ✓"이 표시되는지 확인하세요.
연결이 끊긴 경우 "Google Drive 연결" 버튼을 다시 클릭해 재연결합니다.
Q
확장 프로그램을 업데이트하려면?
담당자에게 새 파일을 받아 기존 폴더에 덮어씌운 뒤,
chrome://extensions에서 캡처 프로그램의 새로고침(↺) 버튼을 클릭하면 됩니다.
Q
웹앱에 데이터가 사라졌어요.
스케줄·묶음 데이터는 각 PC의 확장 프로그램에 저장됩니다.
다른 PC에서 접속하면 해당 PC에 등록된 데이터가 표시됩니다. PC별로 개별 등록이 필요합니다.
문의사항은 담당자에게 연락해주세요.
